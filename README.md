# RideThing
RideThing is a note taking app that design for mouseless lover and all data is local first, we hope people can be efficient and enjoy note taking and learning.


## Feature
* Vim mode
* Label as catelog
* Block Link
* Sync to Github
* Integrate excalidraw
* Concept graph
* Intelligent suggestion
