# Thunder

## Purpose

Help people to organize knowleage and abstract models base on notes. it will find connections between knowledages by algorithm.

[DEMO](https://thunder-web.vercel.app/)

## Feature
* Vim mode
* Label as catelog
* Block Link
* Sync to Github
* Integrate excalidraw
* Concept graph
