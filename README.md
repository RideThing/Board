# Thunder

## Purpose

Help people organize knowleage and abstract models and links from
them. Throught active recall knowleage people will keep knowledge in mind

[DEMO](https://thunder-web.vercel.app/)

## Feature
* Vim mode
* Label as catelog
* Block Link
* Sync to Github
* Integrate excalidraw
* Concept graph
