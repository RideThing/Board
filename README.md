# Thunder

An application for organizing your knowleage and help you abstract models and links from
them. Throught active recall knowleage people will keep knowledges in mind

## Feature
* Vim mode
* Label as catelog
* Block Link
* Sync to Github
* Integrate excalidraw
* Concept graph
