# Thunder
Thunder is a mouseless WYSIWYG edtior, try to help people to organize knowleage, find connections and abstract models

[DEMO](https://thunder-web.vercel.app/)

## Feature
* Vim mode
* Label as catelog
* Block Link
* Sync to Github
* Integrate excalidraw
* Concept graph
* Intelligent suggestion
